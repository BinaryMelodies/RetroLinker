Detailed Description

A record that defines a data type, used for TYPDEF records.

Intel defines types as a sequence of leaf descriptors, each element of a leaf encoding an integer, a type index, a string etc. In the OMF file they are stored as groups of 8, but what is relevant is that it is a sequence of items. The first entry is usually a number that represents what kind of type this is, and the following items describe the parameters to this type.
